5. Panel de Administración del Blog

El sitio incluye un panel de administración protegido con contraseña para gestionar los artículos del blog. No se necesitan conocimientos técnicos para usarlo.

Acceso al Panel

URL giamamg.com/admin
Contraseña (configurar en Vercel: ADMIN_PASSWORD)

Cómo publicar un artículo

  1. Ingresar a giamamg.com/admin con la contraseña
  2. Hacer clic en "+ Nuevo artículo"
  3. Completar el formulario:
    • Título: nombre del artículo
    • Extracto: resumen corto que aparece en la tarjeta del blog
    • Imagen: subir una foto de portada
    • Contenido: escribir el texto separando cada párrafo con una línea en blanco
    • Autor: nombre (por defecto "GIAMA")
    • Fecha: fecha de publicación
  4. Hacer clic en "Publicar artículo"

El artículo aparece en el blog en menos de 1 minuto.

Editar o eliminar un artículo

Desde el panel, cada artículo tiene botones para Ver, Editar o Eliminar.

6. Configuración Inicial en Vercel

Para que el panel de administración del blog funcione, se necesita configurar lo siguiente una única vez desde el dashboard de Vercel:

Paso 1: Crear base de datos Postgres

  1. Ir a Vercel Dashboard > Storage > Create Database > Postgres
  2. Conectar la base al proyecto GIAMA
  3. Ir a la pestaña "Query" y ejecutar el SQL del archivo setup-db.sql

Paso 2: Crear almacenamiento de imágenes (Blob)

  1. Ir a Vercel Dashboard > Storage > Create Store > Blob
  2. Conectar al proyecto GIAMA

Paso 3: Agregar variables de entorno

Ir a Settings > Environment Variables del proyecto y agregar:

VariableDescripción
ADMIN_PASSWORDContraseña para el panel admin (elegir una segura)
ADMIN_SECRETClave secreta para firmar sesiones (cadena de 32+ caracteres)
Importante: Después de agregar las variables y crear la base de datos, hacer un nuevo deploy del proyecto para que los cambios tomen efecto.
